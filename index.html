Dev Tinder

DevTinder is a full-stack developer networking platform built using the MERN stack. The idea behind this project was to solve the problem of fragmented tech communities by creating a centralized space where developers can connect, collaborate, and grow together. Users can create detailed profiles, send and receive smart connection requests, browse a personalized developer feed, and engage in real-time conversations powered by WebSockets. On the backend, I implemented secure authentication using JWT, optimized MongoDB with pagination and indexing for faster queries, and designed scalable schemas with proper validation. The platform is built with performance and real-world scalability in mind.
